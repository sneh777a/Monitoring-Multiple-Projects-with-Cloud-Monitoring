# 📊 Monitoring Multiple Projects with Cloud Monitoring 🚀  
# Cloud Monitoring Practice (GCP)

## 📘 About  
This repository contains my hands-on practice while learning **Cloud Monitoring in Google Cloud Platform (GCP)**.

It focuses on monitoring **multiple projects from a single centralized setup** using Metrics Scope and observability tools.

---

## 🎯 Objective  

- Understand multi-project monitoring in GCP  
- Learn how to use Metrics Scope  
- Practice monitoring resources across projects  
- Implement uptime checks and alerting  

---

## 🛠️ What I Practiced  

- Creating and configuring Metrics Scope  
- Adding multiple projects to a monitoring workspace  
- Monitoring VM instances across projects  
- Setting up uptime checks  
- Creating alerting policies  
- Building custom dashboards  

---

## 🚀 Run the Automation Script  

You can run the monitoring setup script directly from **Google Cloud Shell**:

```bash
curl -LO https://raw.githubusercontent.com/sneh777a/Monitoring-Multiple-Projects-with-Cloud-Monitoring/265ee9652bb7a8be43a04064bb32c1ea84925e19/gcp-monitoring-setup.sh
chmod +x gcp-monitoring-setup.sh
./gcp-monitoring-setup.sh
```
